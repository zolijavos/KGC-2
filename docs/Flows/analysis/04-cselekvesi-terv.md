## Felelősségi Mátrix

### RACI Mátrix

| Feladat | KGC | My Forge | Holding | Ügyvéd |
|---------|-----|----------|---------|--------|
| Folyamatok dokumentálása | **R** | C | I | - |
| Szoftver fejlesztés | C | **R** | A | - |
| Holding alapítás | **R** | R | A | **R** |
| Marketing stratégia | C | C | **R** | - |
| Franchise dokumentáció | C | C | **R** | C |
| Jogi dokumentumok | I | I | A | **R** |
| Technológiai döntések | C | **R** | A | - |

**R** = Responsible (Felelős), **A** = Accountable (Számon kérhető), **C** = Consulted (Konzultált), **I** = Informed (Tájékoztatott)
