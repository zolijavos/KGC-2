## 3. Javasolt Modell: Holding

### 3.1 Tulajdonosi Struktúra

```
┌─────────────────────────────────────────────────────────────────┐
│                    TULAJDONOSI ARÁNYOK                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Holding tulajdonosai:                                          │
│                                                                  │
│  ┌─────────────────┐       ┌─────────────────┐                  │
│  │    KGC Cég      │       │   My Forge      │                  │
│  │                 │       │                 │                  │
│  │     50%         │       │     50%         │                  │
│  │  (vagy 51-49)   │       │  (vagy 49-51)   │                  │
│  └─────────────────┘       └─────────────────┘                  │
│                                                                  │
│  Alternatív opciók:                                             │
│  • 60-40 (KGC többség, domain tudás elismerése)                │
│  • 51-49 (egyértelmű döntéshozó)                                │
│  • 50-50 + tie-breaker mechanizmus                              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Döntéshozatali Mechanizmus

| Döntés típusa | Ki dönt | Szavazati arány |
|---------------|---------|-----------------|
| **Napi operatív** | Ügyvezetés | Egyszerű |
| **Stratégiai** | Tulajdonosok | Egyhangú |
| **Befektetés (< X Ft)** | Ügyvezetés | Egyszerű |
| **Befektetés (> X Ft)** | Tulajdonosok | 75%+ |
| **Új leányvállalat** | Tulajdonosok | Egyhangú |
| **Cégeladás** | Tulajdonosok | Egyhangú |

### 3.3 Patthelyzet Kezelése

```
Ha nincs egyetértés (50-50 esetén):

1. Mediáció
   ├── Belső egyeztetés (7 nap)
   └── Külső mediátor (14 nap)

2. Ha sikertelen:
   ├── Vételi jog (egyik fél megveheti a másikat)
   ├── Eladási jog (harmadik félnek)
   └── Felszámolás (végső esetben)
```

## 7. Kilépési Opciók

### 7.1 Exit Stratégiák

| Szcenárió | Mechanizmus |
|-----------|-------------|
| **Egyik fél ki akar lépni** | Vételi jog a másik félnek |
| **Külső befektető** | Közös döntés, elővásárlási jog |
| **Teljes eladás** | Közös döntés, pro-rata elosztás |
| **Vita esetén** | Mediáció → Arbitráció |

### 7.2 Értékelés

```
Cégeladás esetén az értékelés alapja:
• Bevétel szorzó (SaaS esetén: 3-5x ARR)
• EBITDA szorzó
• Ügyfélszám
• Növekedési ráta
• Független értékbecslő
```
