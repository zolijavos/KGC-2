## Megbeszélés Főbb Pontjai

### 1. Stratégiai Együttműködés

A felek megállapodtak egy hosszú távú stratégiai partnerségben, amelynek célja:
- **KGC ERP rendszer** fejlesztése és üzemeltetése
- **Holding struktúra** kialakítása közös tulajdonláshoz
- **Franchise modell** kidolgozása országos terjeszkedéshez
- **Technológiai modernizáció** (felhő, AI, automatizálás)

### 2. Vállalati Struktúra Koncepció

            ┌─────────────────┐     ┌─────────────────┐
            │   KGC Cég       │     │  My Forge Labs  │

### 3. Üzleti Modell

| Elem | Leírás |
|------|--------|
| **Szoftver** | SaaS modell (bérlés, nem eladás) |
| **Franchise** | Díj alapú, önálló pénzügyi működés |
| **Központi szolgáltatások** | Marketing, IT, beszerzés |
| **Bevételi források** | Franchise díj, szoftver bérlés, központi beszerzés árrés |

### 4. Technológiai Irányok

| Döntés | Választás | Megjegyzés |
|--------|-----------|------------|

| **Offline mód** | Szükséges | PWA, lokális sync |
| **AI integráció** | Igen | E-számla parsing, árazás |

## Azonnali Teendők

### Rövid táv (1-2 hét)

1. **Együttműködési szerződés** - Megbízási szerződés vagy feles modell
2. **Kommunikációs csatornák** - Discord szerver beállítása
3. **Dokumentáció** - Folyamatok átadása, notebook elemek megosztása
4. **Ügyvédi konzultáció** - Holding struktúra jogi háttere

### Közép táv (1-3 hónap)

1. **Holding alapítás** - Jogi személy létrehozása
2. **Szoftver fejlesztés** - MVP elkezdése
3. **Marketing stratégia** - Központi marketing terv
4. **Franchise dokumentáció** - Szerződésminták, szabályzatok
